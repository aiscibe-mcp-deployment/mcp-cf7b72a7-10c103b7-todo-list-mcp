# Todo List MCP

MCP (Model Context Protocol) Server

## Deployment

This MCP server is automatically deployed to Render.com.

## Usage

Connect to this MCP server using:
- Claude Desktop
- MCP Inspector
- Any MCP-compatible client

## Environment Variables

Set these in your Render dashboard if needed:
- `PORT` - Server port (set automatically by Render)

## Development

```bash
pip install -r requirements.txt
python main.py
```
